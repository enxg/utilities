# @enxg/ts-config

**TypeScript configuration used by my projects.**

</div>

## Usage

### Base Config

```json
{
	"extends": "@enxg/ts-config"
}
```

### Config without decorators

```json
{
	"extends": "@enxg/ts-config/without-decorators"
}
```

### Config for NuxtJS 3 projects

```json
{
	"extends": "@enxg/ts-config/nuxt"
}
```
